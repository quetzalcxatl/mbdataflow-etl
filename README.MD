# MBDataFlow_ETL

ETL/EL pipelines for **MetrobГєs CDMX** operational data, deployed as **Google Cloud Run Jobs**.

> **Status:** Work in progress. Several pipelines are functional; others are under active development.

## Overview

This monorepo contains data pipelines that extract operational data from MetrbГєs sources, transform it as needed, and load it into **Google BigQuery** and **Google Drive** for analysis and reporting. Each pipeline targets a specific datadomain (vehicle fleet, CAN bus telemetry, operator reports, vehicle decomissioning, etc.) and is desinged to run as an independent Cloud Run Job.

## Tech Stack

- **Python 3.13**
- **Google Cloud Platform** - Cloud Run Jobs, Cloud Scheduler, BigQuery, Secret Manager
- **Google APIs** - Drive, Sheets (via `google-api-python-client`)
- **Data** - `pandas`, `numpy`
- **Web automation** - `selenium`
